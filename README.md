# B2B cash limited TEST

## Install
1. pull repo
2. ``composer install``
3. copy env.example to .env
4. set db config to .env
5. run migrations ``vendor/bin/phinx migrate``
6. run `cd public` and `php -S localhost:8000`
7. test api via `localhost:8000`
